Remove a dependency from every package that depends on it

## Filter workspaces

**`--only [name glob]`**

Filter workspaces by name

**`--ignore [name glob]`**

Filter out workspaces by name

**`--only-fs [file glob]`**

Filter workspaces by file path

**`--ignore-fs [file glob]`**

Filter out workspaces by file path
