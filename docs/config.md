---

### **`bolt config delete <key>`**

Removes the given `key` from the config.

### **`bolt config list`**

Displays the current configuration.

### **`bolt config get <key>`**

Prints the value for a given `key`.

### **`bolt config set <key> <value> [-g|--global]`**

Sets the config `key` to the given `value`.
